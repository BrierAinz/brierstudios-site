Project Templates

Start fast. Each template scaffolds a complete Yggdrasil CLI project with structure, deps, and examples.

ᛟ

Othala — Static Site

Vanilla HTML/CSS/JS, deploy to Cloudflare Pages, CI/CD ready. Perfect for landing pages, blogs, portfolios.

ygg init my-site --template othala
ᛏ

Tiwaz — CLI Tool

Node.js + Ink for terminal UIs. Slash commands, plugin architecture, easy distribution via npm.

ygg init my-cli --template tiwaz
ᚦ

Thurisaz — AI Agent

Python + pydantic-ai + FastMCP. Multi-agent orchestration, tool definitions, structured outputs.

ygg init my-agent --template thurisaz
ᚠ

Fehu — Brand Kit

Logo variants, color tokens, typography rules, voice guidelines. For studios and creators.

ygg init my-brand --template fehu
ᚹ

Wunjo — Discord Bot

discord.js + slash commands + lore-aware responses. Drop into your server.

ygg init my-bot --template wunjo
ᛊ

Sowilo — Documentation

Docusaurus v3 + Norse theme. MDX, search, dark mode out of the box.

ygg init my-docs --template sowilo
